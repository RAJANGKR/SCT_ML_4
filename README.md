# Hand Gesture Recognition

This project was completed as **Task 4** of the **SkillCraft Technology Machine Learning Internship**.
The objective is to build a hand gesture recognition model that can classify different hand gestures from image data for gesture-based interaction systems.[1]

## Project Overview

The implementation uses the **LeapGestRecog** hand gesture dataset from Kaggle and trains a Convolutional Neural Network (CNN) in **Google Colab**.[2]
To keep training practical, a smaller subset of the dataset was created and used for model development and evaluation.[2]

## Workflow

- Download dataset from Kaggle using the Kaggle API.[2]
- Create a smaller train/validation subset.[2]
- Preprocess images using resizing, grayscale conversion, and normalization.[2]
- Train a CNN model for multi-class hand gesture classification.[2]
- Evaluate performance using accuracy, classification report, and confusion matrix.[3]

## Tech Stack

## Outcome

This project demonstrates a complete computer vision workflow, from dataset preparation to model training and evaluation, while keeping the implementation lightweight enough for internship submission purposes.[3][2]
